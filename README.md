# MAA (unlock)

**MAA (unlock)** is a free, unlocked version of MaceAttackAssistance for Minecraft Fabric 1.21.11

All features enabled — no server authorization required.

## Features
- Aim Assist
- Auto Elytra Swap
- Breach Swap
- Camera Lock
- Stun Slam
- Spear Assist
- Wind Charge throw
- And more...

## Installation
1. Download the latest `.jar` from [Releases](../../releases)
2. Drop it into your `.minecraft/mods` folder
3. Requires Fabric Loader `>=0.18.3` and Fabric API

## Version
- Minecraft: `1.21.11`
- Fabric Loader: `>=0.18.3`
- MAA version: `4.0.5-beta.3`
